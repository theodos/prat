Primer Checker Analysis Tool is a tool to help you analyse the results from Primer Checker at GISAID.

It accepts as input a Primer Checker results file containing mutations for forward/reverse primer and probe. One must also input the total number of COVID-19 variants Primer Checker used.

It outputs tables and graphs you can also use in publications presneting the information from the PRimer Checker about COVID-19

You start your analysis by selecting the Start tab

P.R.A.T. (Primer Checker Results Analysis Tool) for SARS-CoV-2 is a tool to help you analyse the GISAID’s Primer Checker results about SARS-CoV-2.

It uses as input a Primer Checker results file containing mutations for forward/reverse primer and probe. One must also input the total number of SARS-CoV-2 variants Primer Checker used.

The results is a series of tables and graphs that can be used in publications presneting the information from the Primer Checker about SARS-CoV-2.

Start Analysis

You start your analysis by selecting the Start tab

Here the user
  1. must upload the result file obtained from GISAID. The provided file, already obtained from GISAID can be used as an example.
  2. must specify the number of the variants. (Unfortunately, the file itself does not contain this information).
  3. may, optionally, specify a date range.
    4. When field are filled, the analysis takes place automatically. At its end, the Results tab can be selected.

Reviewing the results

By selecting the Results tab the user moves to the Results of the Analysis of the input file he has provided. The results are split in 5 tabs:
  • Original Data

    This is an overview of the input GISAID file. All the alignments of the query file used in GISAID are shown here (hence “Original Data”). Here the user may see all alignments of the query sequences with the sequences deposited in GISAID along with any mismatches. For each alignment there is more information that can be searched through the filtering fields at the top of the list. Additionally, the results may be sorted according to any column.

  • Filter parameters

    The filter parameters allow for further customization of the analysis results. For the time being the user may specify a more restrictive Date Range or restrict the results to one or more specific countries. The results shown in the ‘Original Data’ column won’t change but the results analysed in the following tabs will.

  • Statistics
    • table 1:

      the number of cumulative mutations, as a total and as a percentage.

    • table 2, Position of Single Mutations: The position of each mutation and the mutated nucleotide are shown.

    • table 3, Detailed Single Mutations for forward primer type:

      the cumulative number of mutations in each position and for each nucleotide for the forward primer.

    • table 4, Detailed Single Mutations for rev primer type:

      Same as table 3, for the reverse primer.

  • Graph for fwd primer

    The graph for the forward primer can be customized. More specifically, one can customize:

    • each nucleotide color
    • the Main graph title and the 2 axis labels
    • the Cut-off Percent value. Only labels above the cutoff value are shown in the graph.
    After customization, the user may download the graph in a high-quality pdf,jpg, png or tiff format.
